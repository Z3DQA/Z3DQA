РВ-1. Нет возможности указать в профиле адрес доставки
Важность: Высокая
Окружение: Windows 10, Yandex Browser
Шаги:
1. Открыть сайт kazan.rvkusa.com
2. Зайти в личный кабинет
3. Проскроллить вниз до конца
ФР: В поле адреса доставки нет возможности ввести данные
ОР: В поле адреса доставки есть возможность указать адрес
Приоритет: Высокий



РВ-2. Выполнение условий акции "Закажихиты"нарушено
Важность: Высокая
Окружение: Windows 10, Yandex Browser
Шаги воспроизведения:
1. Открыть сайт kazan.rvkusa.com
2. Ввести промокод(333) в поле для акций
3. Добавить в корзину любые товары для набора суммы заказа >600 рублей
ФР: Возможность заказа на сумму большую 600 рублей отсутсвует
ОР: Есть возможность заказать на сумму больше 600 рублей для выполнения акции
Приоритет: Высокий
